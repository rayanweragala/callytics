# Call Logs

Call Logs provide a searchable history of calls handled by Callytics. Operators and admins can filter by direction, status, and search terms, then open a specific call to understand what happened from start to finish.

Each call can show the execution trace of the flow, including every step the caller visited. This makes it easier to answer questions like which menu option was pressed, whether a queue timed out, or where a call was transferred.

For quality troubleshooting, Callytics scores RTP metrics and presents them with clear labels and color-coded badges. Active calls can also be followed from a live execution panel, and SIP-related issues can be opened directly in SIP Capture using the call's Call-ID.

## Capabilities

- Searchable paginated call history
- Direction filters
- Status filters
- Clear filters action
- Per-call execution trace showing every node visited
- RTP quality scoring with MOS, jitter, packet loss, and RTT
- Color-coded quality badges
- Quality drawer with plain-English labels
- Deep link to SIP Capture filtered by Call-ID
- Live execution panel for active calls
