# Queues & Operators

Queues & Operators provide structured call distribution for teams that answer shared inbound calls. Callers can wait in a queue while Callytics connects them to available operators based on the queue setup.

Operators can log in from a softphone by entering their PIN, making queue availability easy to control without opening the admin UI. The system tracks whether operators are available, busy, or offline so supervisors can understand staffing in real time.

Queue handling also covers practical call-center outcomes such as callers abandoning the queue or reaching a timeout. The live dashboard gives visibility into queue activity, waiting callers, and operator status while calls are in progress.

## Capabilities

- Queue-based inbound call distribution
- Operator PIN login from softphone keypresses
- Real-time operator status: available, busy, or offline
- Caller wait handling when no operator is free
- Abandoned-call routing
- Timeout routing
- Live dashboard visibility for queues and operators
