# Feature Overview

Callytics provides a comprehensive suite of tools for managing a modern call center. Below is a summary of the core features.

| Feature | Description |
|---|---|
| **IVR Flow Builder** | Drag-and-drop canvas for building complex call logic without writing code. |
| **Queues & Operators** | Group extensions into queues for distributed call handling. |
| **Outbound Dialer** | Run scheduled campaigns with CSV contact lists and automated retries. |
| **WireGuard VPN** | Integrated VPN server for secure, NAT-traversing remote SIP access. |
| **SIP Firewall** | Pattern-based registration abuse detection and automatic blocking. |
| **Backup & Restore** | Scheduled snapshots of configuration, database, and call recordings. |
| **Live Dashboard** | Real-time overview of active calls, queue stats, and system health. |
| **SIP Capture** | Packet-level SIP troubleshooting directly from the browser. |
| **Reports** | Volume, missed calls, and performance reports with CSV export. |
| **Audio Library** | Manage MP3/WAV uploads and generate prompts with offline TTS. |
