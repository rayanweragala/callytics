# Features

## Flow Builder

The Flow Builder provides a drag-and-drop canvas for creating, editing, versioning, and publishing call flows with real-time persistence and validation.

## Node Types

The node system includes 13 production node types that cover entry, routing, media playback, menus, transfer, queueing, voicemail, recording, variables, callbacks, conferencing, and termination behaviors.

## SIP Extensions

Extension management supports create, edit, delete, QR provisioning data, and backend-driven PJSIP config regeneration for database-backed SIP users.

## SIP Trunks

Trunk management supports provider credentials, enable/disable state, connectivity tests, and generated trunk config reloads through managed backend workflows.

## Inbound Routing

Inbound DID routing maps called numbers to target flows so incoming calls are directed into the correct runtime flow graph without manual dialplan edits.

## Audio Management

Audio management supports file upload, server-side conversion, searchable assets, browser preview, and reusable prompt assignment across flow nodes.

## TTS

Offline Piper text-to-speech generation lets operators create prompts locally with no cloud dependency, then store them directly in the audio library.

## Recordings

Call recordings are captured by the runtime, indexed in the UI, and available for inline playback or direct file download.

## Live Dashboard

The dashboard streams active calls, queue status, and recent runtime events so operators can monitor live traffic and system behavior.

## Call Logs

Call logs provide searchable historical records with execution trace views and per-call quality indicators to support operational and support analysis.

## SIP Capture

SIP Capture provides live packet visibility, historical packet retrieval, ladder rendering, and `.pcap` export for deep SIP troubleshooting.

## Diagnostics

Diagnostics includes trunk health checks, SIP registration views, failure insights, and host resource telemetry for operational verification.

## Asterisk Logs

Asterisk log tooling reads runtime log output and presents plain-English translation cues for faster incident triage.

## Outbound Campaigns

Outbound campaigns support campaign records, CSV contact uploads, scheduling, progress tracking, and controlled sliding-window dialing.

## Queues and Operators

Queue and operator modules provide queue definitions, operator assignment, and queue-based call distribution for team handling scenarios.

## Hunt Groups

Hunt group routing supports sequential, random, and parallel group strategies with retry and no-answer branch behavior.

## Conference Rooms

Conference room flows use ConfBridge-backed rooms for multi-party calling with moderator-aware behavior.

## Callbacks

Callback workflows allow callers to request a callback and feed callback jobs into managed outbound execution paths.

## WireGuard VPN

WireGuard VPN features provide peer provisioning, QR onboarding, status visibility, and secure remote endpoint access controls.

## SIP Firewall

The SIP firewall monitors abuse signals, applies threshold-based blocking, and provides live security feed visibility with optional GeoIP context.

## Backup and Restore

Backup and restore supports manual/scheduled backups, retention policy configuration, archive downloads, and restore execution from uploaded packages.

## Templates

Template tools provide one-click IVR template imports and JSON-based flow import paths to accelerate new deployments.

## Preflight Wizard

The preflight wizard runs readiness checks across telephony and infrastructure prerequisites before live production usage.
